# Taskify

Aplicacao web Single Page para gestao de tarefas (Todo List), desenvolvida para o exercicio de Frontend.

## Visao geral

Taskify permite criar, listar, editar, concluir e apagar tarefas com uma interface moderna, animacoes e persistencia local.

## Funcionalidades

- Single page website (sem navegacao por multiplas paginas)
- CRUD completo de tarefas
- Ler todas as tarefas (GET logico)
- Criar tarefa (POST logico)
- Editar tarefa (PUT logico)
- Apagar tarefa (DELETE logico)
- Persistencia em localStorage
- Registo de data e hora de criacao por tarefa (DateTime API)
- Notificacoes com SweetAlert2 para operacoes de criar, editar e apagar
- Icones nos botoes de acao
- Interface customizada com CSS proprio (sem biblioteca de UI)
- Layout responsivo para desktop e mobile

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)
- SweetAlert2
- AOS (Animate On Scroll)
- GSAP

## Estrutura do projeto

- FINAL/index.html: Estrutura da interface
- FINAL/style.css: Estilizacao e responsividade
- FINAL/script.js: Logica da aplicacao e operacoes CRUD
- FINAL/media.css: Ajustes adicionais de media queries

## Como executar localmente

1. Abrir a pasta do repositorio no VS Code.
2. Abrir o ficheiro FINAL/index.html no browser.
3. Interagir com a app e validar o CRUD.
